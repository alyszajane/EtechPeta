What is HTML?

HTML stands for HyperText Markup Language. It is the standard language for creating web pages.

Basic Structure of an HTML Document

                
<!DOCTYPE html>
<html>
<head>
<title>My First HTML Page</title>
</head>
<body>
<p>This is my first HTML page.</p>
</body>
</html>

HTML Elements

HTML elements are the building blocks of HTML pages. They are represented by tags.

Common HTML Tags

  • <h1> to <h6> - Headings
  • <p> - Paragraph
  • <a> - Anchor (link)
  • <img> - Image
  • <div> - Division or section
  • <span> - Inline section

Example of Headings

                
<h1>This is a Heading 1</h1>
<h2>This is a Heading 2</h2>                 
<h3>This is a Heading 3</h3>

HTML Attributes

Attributes provide additional information about HTML elements. They are always specified in the opening tag.

Common Attributes

  • href - Specifies the URL for links.
  • src - Specifies the source of an image.
  • alt - Provides alternative text for images.
  • title - Adds extra information about an element.

Example

                
<a href="https://www.example.com" title="Visit Example">Visit Example</a>

HTML Forms

Forms are used to collect user input. They can contain various types of input elements.

Basic Structure of a Form

                
<form action="submit.php" method="post">
<label for="name">Name:</label>
<input type="text" id="name" name="name">
<input type="submit" value="Submit">
</form>

HTML Tables

Tables are used to display data in a structured format. They consist of rows and columns.

Basic Structure of a Table

                
<table>
<tr>
<th> Header 1</th>
<th>Header 2</th>
</tr>
<tr>
<td>Data 1</td>
<td>Data 2</td>
</tr>
</table>

Adding Images

To add an image to your webpage, use the <img> tag. The src attribute specifies the path to the image, and the alt attribute provides alternative text for the image.

Example of Adding an Image

                
<img src="image.jpg" alt="Description of image" width="300" height="200">

Adding Audio

To add audio to your webpage, use the <audio> tag. You can specify controls to allow users to play, pause, and adjust the volume.

Example of Adding Audio

                
<audio controls>
  <source src="audio.mp3" type="audio/mpeg">
  Your browser does not support the audio element.
</audio>

HTML Lists

HTML supports ordered and unordered lists. Use the <ul> tag for unordered lists and the <ol> tag for ordered lists.

Example of Unordered List

                
<ul>
  <li>Item 1</li>
  <li>Item 2</li>
  <li>Item 3</li>
</ul>

Example of Ordered List

                
<ol>
  <li>First Item</li>
  <li>Second Item</li>
  <li>Third Item</li>
</ol>

HTML Links

Links are created using the <a> tag. The href attribute specifies the URL of the page the link goes to.

Example of a Link

                
<a href="https://www.example.com">Visit Example</a>

HTML Iframes

Iframes allow you to embed another HTML page within your current page. Use the <iframe> tag for this purpose.

Example of an Iframe

                
<iframe src="https://www.example.com" width="600" height="400"></iframe>

Semantic HTML

Semantic HTML uses HTML markup to reinforce the meaning of the content. Elements like <header>, <footer>, <article>, and <section> are examples of semantic elements.

Example of Semantic HTML

</header>
